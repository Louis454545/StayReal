# StayReal

A work in progress open-source client for BeReal.

## Features

- [x] Authentication
  - [x] Send OTP
  - [x] Verify OTP
  - [x] Store tokens
  - [x] Refresh token when expired
- [ ] My Profile
  - [x] Informations
  - [x] Realmojis
  - [ ] Pinned
  - [ ] Memories
- [ ] (any) User Profile
- [ ] Feed
  - [x] My posts
  - [ ] Comments on my posts
  - [ ] Swipper for my posts
  - [x] Users
  - [x] Posts
  - [ ] Swipper for multiple posts
  - [ ] Comments
  - [ ] Reactions
- [ ] Post
- [ ] Comment
- [ ] React with realmoji
- [ ] Add friend
- [ ] Friends
