### Prerequisites

- [Node.js](https://nodejs.org/)
- [Rust](https://www.rust-lang.org/)
- [Tauri v2](https://v2.tauri.app/fr/start/prerequisites/)

### Installation

```bash
# Clone the project
git clone https://github.com/Vexcited/StayReal.git

# Install dependencies
cd stayreal
pnpm install

# Launch in development
pnpm run tauri dev
```
